# ガイダンス

[ 目次に戻る

## 概要
講義の概要と進め方について解説し，地理空間データの概要を学ぶ

## ねらい（Learning Objectives）
- 理論の要点を理解する。
- 手順に沿って演習を完了できるようになる。
- 得られた結果を適切に解釈・可視化する。

## 到達目標（Outcomes）
- 基本的な操作・関数・ワークフローを説明できる。
- オープンデータを用いた簡単な分析を自力で再現できる。

## 事前準備（Preparation）
- 貸与ノートPCなどのPC端末を持参すること。
- 講義用GitHub Pagesサイトへアクセスできるインターネット環境。

## 使用ツール・環境（Tools）
- QGIS (導入概要のみ)
- R / RStudio (導入概要のみ)

## 演習の流れ（Procedure）
1. 導入：目的と全体像の確認。
2. セットアップ：必要なプロジェクト・フォルダ・レイヤ／スクリプトの準備。
3. 実行：ステップごとの操作・コード実行。
4. 検証：結果の確認と考察。
5. まとめ：質疑応答と次回への接続。

## 課題（Assignment）
- 授業中の演習を自分の環境で再現し、成果物（スクリーンショット・プロジェクトファイル・Rスクリプト等）を提出。
- 気づきや改善案を100–200字程度で記述。

## 使用データ（Datasets）
- オープンデータ（例：行政界、道路網、地点データなど）。※詳細は当該回で案内。

## 参考資料（References）
- 授業資料（GitHub Pages／配布ドキュメント）。
- 教科書・推奨文献（該当箇所）。
